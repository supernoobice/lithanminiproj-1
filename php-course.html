PHP & MySQL

Hack the back-end!

In this course you will learn to engineer dynamic web applications using PHP and MySQL as its database.

You will learn to develop dynamic websites and web applications using PHP that powers most of the popular sites on the web such as Facebook and Wikipedia.

With the knowledge you will gain in this course, you are ready to take on a career as web developer or become a freelancer.

Objectives

  • To have a strong foundation in web development
  • To have a strong foundation in MySQL
  • To learn how to design scalable databases with MySQL
  • To have students be able to produce their own web application from start to finish
  • To deploy sites on the web
  • To learn to read and write code in PHP

Key Topics

  1. Intro to PHP Web Development
  2. Variables
  3. Mathematical Operations
  4. Logical & Comparison Operations
  5. Arrays
  6. Conditional Statements
  7. Functions
  8. Form Handling & Inputs
  9. Classes & Objects
  10. Object Oriented Programming (OOP)
  11. MySQL Database
  12. PHP & MySQL Database
  13. Deploying Your App on the Web

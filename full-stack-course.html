Full Stock Web Development

From Jobless to Full Stack Developer

One of the most in-demand jobs for the next 10 years.

Learn how to code in front-end and back-end web technologies. In this course you will learn how to take advantage of HTML, CSS and Javascript to create web applications. When you get into the advanced modules, you will learn how to use frameworks like jQuery and Angular to build front-end apps. And using Java Struts to develop the backend server.

Objectives

  • To have a strong foundation in web development
  • To have a strong foundation in Databases
  • To learn how to design scalable databases with MySQL
  • To have students be able to produce their own web application from start to finish
  • To deploy sites on the web
  • To learn to read and write code in Java

Key Topics

  1. Module 1: Basic Java
  2. Module 2: HTML, CSS and jQuery
  3. Module 3: Front-end Development using Angular
  4. Module 4: Back-end Development using Java Struts
